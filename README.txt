-----------STEPS--------------(FRAMEWORK: REACT, MICROSOFT VISUAL STUDIO CODE)

1.Clone the Repository:
  git clone https://github.com/sahulrao/star-wars-planets-directory.git
  Navigate to the Project Directory:

2.cd star-wars-planets-directory

3.Installation
  Install Dependencies:
  npm install
  Running the Application

4.Start the Development Server:
  npm start

5.View the Application:
  Open your web browser and go to http://localhost:3000

That's it! You should now have the Star Wars Planets Directory running on your local machine.
